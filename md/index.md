## Overview 

These written materials are designed as part of a short course that teaches survival (time-to-event) analysis in an applied manner.

These materials show how to implement time-to-event analysis in R and/or Python, using life sciences datasets. 

::: {.callout-tip}
### Learning Objectives

- Recognise key features of time-to-event data (e.g., censoring), and the need for specialist modelling techniques
- Visualise time-to-event data with survival curves
- Run significance tests (log-rank test, Cox regression) to assess the impact of predictor variables on survival times
- Interpret the output of significance tests thoughtfully
:::


### Target Audience

The course is aimed at researchers, particularly those in the life sciences, who are looking to analyse their own time-to-event data. 


### Prerequisites

The course assumes a working knowledge of R or Python, as well as an understanding of core statistical topics (up to and including linear modelling).
